# Reto Técnico – ASP.NET MVC

Proyecto desarrollado como parte de una prueba técnica.

## Tecnologías
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server

## Ejecución del proyecto
1. Clonar el repositorio
2. Abrir la solución en Visual Studio
3. Restaurar paquetes NuGet
4. Configurar la cadena de conexión en `Web.config`
5. Ejecutar el proyecto con IIS Express

## Base de datos
El script de base de datos se encuentra en la carpeta `BD`.
