# Reto Técnico – ASP.NET MVC

Proyecto desarrollado como parte de una prueba técnica.

## Tecnologías
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server

## Configuración
1. Restaurar paquetes NuGet
2. Configurar cadena de conexión en `Web.config`
3. Ejecutar el proyecto en IIS Express
